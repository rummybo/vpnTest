常用导航模块

常用导航管理功能，支持完整的 CRUD 操作，包括新增、编辑、删除、排序等功能。

GET /api/v1/{secure_path}/common_links/fetch
获取常用导航列表

请求参数

参数类型必填说明
currentnumber否当前页码，默认1
pageSizenumber否每页数量，默认10
sortstring否排序字段，默认weigh
sort_typestring否排序方式：asc/desc，默认desc
filterarray否筛选条件数组

响应示例

{
  "data": [
    {
      "id": 1,
      "title": "Google",
      "url": "https://www.google.com",
      "logo": "https://www.google.com/favicon.ico",
      "weigh": 100,
      "status": "normal",
      "createtime": 1726387200,
      "updatetime": 1726387200
    }
  ],
  "total": 1
}
POST /api/v1/{secure_path}/common_links/save
保存常用导航（新增/编辑）

请求参数

参数类型必填说明
idnumber否编辑时需要传入ID
titlestring是导航标题，最大100字符
urlstring是链接地址，最大255字符
logostring否图标URL，最大255字符
weighnumber否权重，数值越大排序越靠前
statusstring是状态：normal/hidden
POST /api/v1/{secure_path}/common_links/drop
删除常用导航

请求参数

参数类型必填说明
idnumber是要删除的导航ID
GET /api/v1/{secure_path}/common_links/show
显示/隐藏常用导航

请求参数

参数类型必填说明
idnumber是导航ID
statusstring是状态：normal/hidden
POST /api/v1/{secure_path}/common_links/sort
批量更新排序

请求参数

参数类型必填说明
sortsarray是排序数组，格式：[{"id": 1, "weigh": 100}]
POST /api/v1/{secure_path}/upload/image
上传图片文件

请求参数

参数类型必填说明
filefile是图片文件，支持JPG/PNG/GIF/SVG，最大2MB

响应示例

{
  "message": "图片上传成功",
  "url": "http://192.197.113.52:2053/uploads/nav_links/1726387200_abc123.jpg",
  "path": "uploads/nav_links/1726387200_abc123.jpg",
  "filename": "1726387200_abc123.jpg",
  "size": 102400
}

重要说明

  • 权重排序：weigh 字段数值越大，排序越靠前（降序排列）
  • 状态管理：normal=显示，hidden=隐藏
  • URL自动处理：如果URL没有协议前缀，系统会自动添加https://
  • 图片存储：上传的图片保存在public/uploads/nav_links/目录
  • 认证要求：所有接口都需要JWT认证，在请求头中包含Authorization
常见错误码
400 Bad Request
请求参数错误或格式不正确
401 Unauthorized
未授权访问，需要有效的认证令牌
403 Forbidden
权限不足，无法访问该资源
404 Not Found
请求的资源不存在
422 Unprocessable Entity
请求格式正确但包含语义错误
500 Internal Server Error
服务器内部错误
SMS_SEND_FAILED
短信发送失败
SMS_CODE_INVALID
验证码无效或已过期
