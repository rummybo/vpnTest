用户认证 API

说明： 用户认证接口支持邮箱、用户名、手机号三种方式进行注册和登录。
POST /passport/auth/register

描述： 用户注册接口，支持邮箱、用户名、手机号三种注册方式

请求参数

参数名 类型 必填 说明
email string 可选 邮箱地址（邮箱注册时必填）
username string 可选 用户名（用户名注册时必填）
phone string 可选 手机号（手机注册时必填）
password string 必填 密码
invite_code string 可选 邀请码
email_code string 可选 邮箱验证码（开启邮箱验证时必填）

请求示例

邮箱注册
用户名注册
手机注册
POST http://192.197.113.52:2053/api/v1/passport/auth/register Content-Type: application/json { "email": "user@example.com", "password": "password123", "invite_code": "ABC123" }
POST http://192.197.113.52:2053/api/v1/passport/auth/register Content-Type: application/json { "username": "myusername", "password": "password123", "invite_code": "ABC123" }
POST http://192.197.113.52:2053/api/v1/passport/auth/register Content-Type: application/json { "phone": "13800138000", "password": "password123", "invite_code": "ABC123" }

响应示例

{ "data": { "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...", "auth_data": "encrypted_auth_data", "is_admin": false } }

请求参数

参数名 类型 必填 说明

email_code string 必填 邮箱验证码
password string 必填 新密码

请求示例

POST http://192.197.113.52:2053/api/v1/passport/auth/forget Content-Type: application/json { "email": "user@example.com", "email_code": "123456", "password": "newpassword123" }

响应示例

POST /passport/auth/forgetByPhone

描述： 通过手机号和短信验证码重置密码

请求参数

参数名 类型 必填 说明
phone string 必填 手机号码（11位数字）
code string 必填 短信验证码（6位数字）
password string 必填 新密码（6-32位字符）

请求示例

POST http://192.197.113.52:2053/api/v1/passport/auth/forgetByPhone Content-Type: application/json { "phone": "13800138000", "code": "123456", "password": "newpassword123" }

响应示例

{ "data": true, "message": "密码重置成功" }

错误响应示例

{ "message": "验证码已过期" }
使用流程：
1. 调用 /sms/send 发送重置密码验证码（type=reset_password）
2. 用户输入收到的验证码和新密码
3. 调用此接口完成密码重置
4. 重置成功后用户可使用新密码登录
安全说明：
• 验证码有效期5分钟
• 手机号必须已注册才能重置密码
• 密码重置成功后会记录操作日志
• 验证码使用后自动失效
POST /passport/auth/changePassword

描述： 用户修改密码，支持邮箱、用户名、手机号三种登录方式

请求参数

参数名 类型 必填 说明
identifier string 必填 登录标识符（邮箱/用户名/手机号）
old_password string 必填 原始密码
new_password string 必填 新密码（6-32位字符）
confirm_password string 必填 确认新密码（必须与new_password一致）

请求示例

POST http://192.197.113.52:2053/api/v1/passport/auth/changePassword Content-Type: application/json { "identifier": "user@example.com", "old_password": "oldpassword123", "new_password": "newpassword456", "confirm_password": "newpassword456" }

响应示例

{ "data": true, "message": "密码修改成功" }

错误响应示例

{ "message": "原始密码错误" } { "message": "用户不存在" } { "message": "The confirm password field must match new password." }
支持的登录方式：
• 邮箱登录： identifier 传入邮箱地址（如：user@example.com）
• 用户名登录： identifier 传入用户名（如：username123）
• 手机号登录： identifier 传入手机号（如：13800138000）
安全说明：
• 系统会自动识别登录方式（邮箱/用户名/手机号）
• 必须提供正确的原始密码才能修改
• 新密码和确认密码必须完全一致
• 密码修改成功后会记录操作日志
• 建议用户修改密码后重新登录

短信验证 API

说明： 短信验证接口用于发送和验证手机短信验证码。
POST /sms/send

描述： 发送短信验证码到指定手机号

请求参数

参数名 类型 必填 说明
phone string 必填 手机号码（11位数字）
type string 可选 短信类型：register（注册，默认）| reset_password（重置密码）

请求示例

注册验证码
重置密码验证码
POST http://192.197.113.52:2053/api/v1/sms/send Content-Type: application/json { "phone": "13800138000", "type": "register" }
POST http://192.197.113.52:2053/api/v1/sms/send Content-Type: application/json { "phone": "13800138000", "type": "reset_password" }

响应示例

{ "code": 1, "msg": "短信发送成功", "data": { "type": "register", "expire_time": 600, "sms_result": { "result": true, "msgid": "1234567890" } } }

错误响应示例

{ "code": 0, "msg": "该手机号未注册" }
注意事项：
• 注册验证码有效期10分钟，重置密码验证码有效期5分钟
• 同一手机号1分钟内只能发送一次验证码
• 重置密码类型需要手机号已注册才能发送
POST /sms/verify

描述： 验证手机短信验证码

请求参数

参数名 类型 必填 说明
phone string 必填 手机号码
code string 必填 6位验证码
type string 可选 短信类型：register（注册，默认）| reset_password（重置密码）

请求示例

POST http://192.197.113.52:2053/api/v1/sms/verify Content-Type: application/json { "phone": "13800138000", "code": "123456", "type": "reset_password" }

响应示例

{ "code": 1, "msg": "验证码验证成功", "data": { "type": "reset_password", "phone": "13800138000" } }

错误响应示例

{ "code": 0, "msg": "验证码错误" }
注意：
• 注册验证码有效期10分钟，重置密码验证码有效期5分钟
• 验证成功后验证码会被自动删除
• 验证码为6位数字
• 需要指定正确的type类型才能验证对应的验证码

推广链接 API

说明： 推广链接系统允许用户生成推广码，邀请新用户获得奖励。支持多级奖励机制和金币兑换功能。
POST /share/generate

描述： 为指定设备生成推广链接和推广码

请求参数

参数名 类型 必填 说明
device_id string 必填 设备唯一标识，最大128字符

请求示例

POST http://192.197.113.52:2053/api/v1/share/generate Content-Type: application/json { "device_id": "device_12345678" }

响应示例

{ "success": true, "data": { "device_id": "device_12345678", "promotion_code": "PROMO123456", "share_url": "http://192.197.113.52:2053/device/promotion-bind.html?code=PROMO123456", "current_referrals": 0, "reward_level": { "daily": 60, "single": 30 }, "coins": 0, "is_vip": false } }

响应字段说明

字段 类型 说明
promotion_code string 推广码
share_url string 推广链接
current_referrals integer 当前推荐人数
reward_level object 奖励等级（daily: 每日时长分钟，single: 单次时长分钟）
coins integer 金币数量
is_vip boolean 是否为VIP用户
POST /share/bind

描述： 新用户绑定推广码，获得基础体验时长

请求参数

参数名 类型 必填 说明
promotion_code string 必填 推广码，最大32字符
user_device_id string 必填 用户设备ID，最大128字符

请求示例

POST http://192.197.113.52:2053/api/v1/share/bind Content-Type: application/json { "promotion_code": "PROMO123456", "user_device_id": "user_device_87654321" }

响应示例

{ "success": true, "message": "绑定成功，您获得了基础体验时长", "data": { "daily_duration": 60, "single_duration": 30, "is_unlimited": false, "promoter_info": { "total_referrals": 1, "reward_level": { "daily": 120, "single": 30 }, "coins": 0 } } }
绑定规则：
• 每个设备只能绑定一次推广码
• 不能绑定自己的推广码
• 新用户绑定后获得60分钟/天的基础体验时长
• 推广者根据推荐人数获得相应奖励等级

POST /share/exchange-vip

描述： 使用金币兑换黄金会员

请求参数

参数名 类型 必填 说明
device_id string 必填 设备ID，最大128字符

请求示例

POST http://192.197.113.52:2053/api/v1/share/exchange-vip Content-Type: application/json { "device_id": "device_12345678" }

响应示例

{ "success": true, "message": "兑换成功，您已成为黄金会员", "data": { "is_vip": true, "remaining_coins": 5 } }
奖励等级说明：
• 推荐1人：每日2小时使用时长
• 推荐2人：每日3小时使用时长
• 推荐3人：单次无限制使用
• 推荐23人：无限制使用 + 开始获得金币奖励
• 金币可用于兑换黄金会员等特权
